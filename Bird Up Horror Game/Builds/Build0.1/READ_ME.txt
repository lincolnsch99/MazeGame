Escape From Bird Version 0.1:
Features : 
 - Simple main menu to Play or Quit the Game.
 - Options button not functional.
 - Upon death, player returns to main menu.
 - Always spawns the "easy" level of maze, which is a 10X10 randomly generated maze, with a slower version of the bird.
 - Bird/Enemy object is simpl a capsule, the model has not been imported yet.
 - Bird/Enemy uses sounds to let the player know if it is looking for or chasing the player.

Changes : 
 - First build, so no changes.

Plans : 
 - Player should be able to choose the difficulty in the menu, possibly also a custom difficulty.
 - Player should be able to pause the game, and have an option to Resume, Quit or Return to Menu.
